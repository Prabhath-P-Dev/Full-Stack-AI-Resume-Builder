# AI Resume Builder

A full-stack AI Resume Builder that helps users create professional resumes with AI assistance. Built using React, Node.js, Express, MongoDB, and the Gemini API.

## Features

- User authentication
- AI-powered resume generation
- Resume editing
- Resume download as PDF
- Responsive user interface

## Tech Stack

### Frontend
- React
- Redux Toolkit
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Gemini API

## Project Structure

```text
client/   - React frontend
server/   - Node.js & Express backend
```

## Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/ai-resume-builder.git
```

### Install dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### Start the application

```bash
# Frontend
npm run dev

# Backend
npm start
```
